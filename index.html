Everything You Need

Powerful features for working with Classic Mac disk images

Create DC42 Images

Create DC42 disk images from folders with full support for data and resource forks.

Convert Formats

Convert between DC42, ISO, DMG, IMG formats with a single click.

Browse Contents

View HFS filesystem contents without extraction. Navigate folders and preview files.

Extract Files

Extract individual files or entire volumes to your Mac, iPhone, or iPad.

Universal App

One app for all your devices. iPhone, iPad, and Mac with native experience.

Open Source

Free and open source. Built with Swift and SwiftUI for the community.

Wide Format Support

DC42Studio supports all major disk image formats used by Classic Mac OS and modern systems.
